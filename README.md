# Go Scrum Api

## Description

Proyecto desarrollado en Nest para la aplicacion GoScrum [https://github.com/RassielMex/next_goscrum]

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
